Smart-X-Styles » smart-details

.smart-details

scss 
.smart-details {
  
}

Defines a container that will layout details in a uniform way.
*Note: This selector may not contain any styles, but it is used to target the list by parent components and by JavaScript.

Author:

  • James Stanger

Example

njk 
{% set detailsClass = 'smart-details' %}
{% include 'smart-details.njk' %}
html compiled 
<section class="smart-details">
  <header class="smart-details--header">
     <h3 class="smart-details--heading">Details</h3>
  </header>
  <div class="smart-details--listContainer">
    <dl class="smart-details--list">
      <div class="smart-details--detail"> <dt class="smart-details--detailKey">Name</dt>
        <dd class="smart-details--detailValue">James</dd>
      </div>
      <div class="smart-details--detail"> <dt class="smart-details--detailKey">Age</dt>
        <dd class="smart-details--detailValue">24</dd>
      </div>
      <div class="smart-details--detail"> <dt class="smart-details--detailKey">Student</dt>
        <dd class="smart-details--detailValue">false</dd>
      </div>
      <div class="smart-details--detail"> <dt class="smart-details--detailKey">Position</dt>
        <dd class="smart-details--detailValue">ITS3</dd>
      </div>
      <div class="smart-details--detail"> <dt class="smart-details--detailKey">MoreInfo</dt>
        <dd class="smart-details--detailValue">more</dd>

  </div>
</section>
