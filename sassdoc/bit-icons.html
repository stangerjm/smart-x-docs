.bit-icon-user

scss 
.bit-icon-user {
  &:after {
    @include icon;
    @include icon-url('user', 20px, 20px);
  }
}

Inserts a user / profile icon after the element.

Author:

  • James Stanger

Example

html 
<span class="bit-icon-user"></span>

Requires

@mixin icon()

@mixin icon-url()
