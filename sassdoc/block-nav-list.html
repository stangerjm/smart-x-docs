.block-navList

scss 
.block-navList {
  
}

Defines a container that will hold the responsive navigation list and the mobile-oriented expand button.
*Note: This selector may not contain any styles, but it is used to target the list by parent components and by JavaScript.

.block-navList--list

scss 
.block-navList--list {
  display: flex;
  flex-direction: row;
  background: lighten($theme-main--dark, 20%);

  margin: 0;
  padding-left: 0;
  z-index: $level-9;

  @media screen and (max-width: $break-med) {
    /* mobile list style */
    /* hide by default */
    display: block;

    /* Translate nav list the width of the nav minus an extra 50px to be just off of the left side of the screen */
    $listWidth: 100%;
    width: $listWidth;
    transform: translateX(-$listWidth - 50);
    transition: transform 600ms ease-in-out;

    position: absolute;
    left: 0;

    /* show when nav-show class is applied and only on medium screen */
    &.nav-show {
      display: flex;
      flex-direction: column;
      transform: none;
    }
  }
}

.block-navList--item

scss 
.block-navList--item {
  list-style: none;
  box-shadow: 0 0 1px inset;
  padding: 3px 0;
  text-align: center;
  position: relative;
  flex: 1 auto;

  &:hover {
    /* apply transitions and border only when not mobile. */
    transition: border-top-width 100ms;
    border-top: 2px groove $theme-secondary--deep;
    margin-top: -2px;

    .bit-dropList {
      /* show any sublists on hover */
      display: block;
    }
  }

  @media screen and (max-width: $break-med) {
    /* resize list items on mobile */
    padding: 5px;
    position: relative;

    &:hover {
      border: none;
      margin: 0;
    }
  }
}

.block-navList--expand

scss 
.block-navList--expand {
  /* hide by default */
  display: none;

  /* style expand button when screen is small */
  @media screen and (max-width: $break-med) {
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;

    border: 2px solid $theme-secondary--light;
    border-radius: 10px;
    width: 50px;
    height: 50px;
  }
}

Styles an expand button to expand the navigation on small screens.
*Note: The expand button will only appear on small screens.
