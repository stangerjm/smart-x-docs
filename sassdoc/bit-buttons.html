.bit-btn

scss 
.bit-btn {
  border-radius: 5px;
  padding: 0 3px;
  min-height: 35px;
  min-width: 60px;
  background: $theme-main--deep;
  border: none;
  color: $theme-white;
  box-sizing: content-box;
  text-align: center;
  text-decoration: none;
}

Sets base styles for a button.

Author:

  • James Stanger

Example

html 
<button class="bit-btn">Click</button>

Requires

$theme-main--deep (Color)

$theme-white (Color)

.bit-btn-clickable

scss 
.bit-btn-clickable {
  @include clickable;
}

Styles clickable button animation.
*Note: This is NOT necessary on elements such as bit-btn-search, bit-btn-add, etc. These already have the functionality built in.

Author:

  • James Stanger

Example

html 
<button type="button" class="bit-btn bit-btn-clickable">Click</button>

Requires

@mixin clickable()

.bit-btn-search

scss 
.bit-btn-search {
  @include clickable;

  &:before {
    @include icon;
    @include icon-url('search')
  }
}

.bit-btn-expand

scss

  @include bit-btn-noBackground;
  @include clickable-shadowless;
  line-height: 1.6;

  &:after {
    @include icon("+");
    font-size: 15px;
    font-weight: 900;
    width: 1.8em;
    height: 1.8em;
    text-align: center;
    border-radius: 1em;
    background: $theme-secondary--deep;
  }
}
