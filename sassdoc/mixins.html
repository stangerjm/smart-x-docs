@mixin icon-url()

Mixin to include an image in an icon element.

Author:

  • James Stanger

Parameters

$iconName: (String)

Name of the icon file.

$width: 15px (Number)

Width of the icon.

$height: 15px (Number)

Height of the icon.

Example

scss Basic use 
 .user-icon {
   @include icon-url("user", 13px);
}
css compiled 
.user-icon {
  background: url("../assets/user.svg") no-repeat;
  width: 13px;
  height: 15px;
}

$img-directory (String)

@mixin icon()

Sets base styles for an icon. Optionally will include a character as the icon.

Author:

  • James Stanger

Parameters

$content: " " (String)

Optional character to use as icon.

Examples

scss Basic use 
.icon {
  @include icon();
}
css compiled 
.icon {
  content: " ";
  display: inline-block;
}
scss Include a character as an icon 
.icon {
  @include icon("*");
}
css compiled 
.icon {
  content: "*";
  display: inline-block;
}

@mixin clickable()

Styles a button to have a click animation.

Author:

  • James Stanger

Example

scss Basic use 
.clickable-btn {
    @include clickable;
}
css compiled 
.clickable-btn {
  box-shadow: 2px 1px 1px #0d0908;
}

.clickable-btn:active {
  box-shadow: 1px 1px 0;
  transform: translate(2px, 2px);
}
