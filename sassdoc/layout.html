.layout-main

scss 
.layout-main {
  /*
      Grid representation:

         1                    2
      1 +--------------------+
        |  NAV     - auto    |
      2 +--------------------+
        |  SEARCH  - auto    |
      3 +--------------------+
        |                    |
        |  CONTENT - fill    |
        |                    |
      4 +--------------------+
        |  FOOTER  - 50px    |
      5 +--------------------+
    */
    grid-template: "nav    " auto
                   "search " auto
                   "content" 1fr
                   "footer " 50px
                   / 1fr;
}

Grid layout for browsers that support CSS Grid
*Note, there is an undocumented fallback for other browsers
