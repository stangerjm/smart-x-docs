.smart-modal

scss 
.smart-modal {
  /* Hide by default */
  display: none;

  position: fixed;
  z-index: $level-100;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  overflow: auto;

  /* fallback color */
  background-color: $theme-black;
  background-color: rgba($theme-black, 0.5);
}

Defines an overlay container that will deny access to the UI while the modal is open.
*Note: this component is hidden by default. The example below is displayed by default for demonstration purposes only.

Author:

  • James Stanger

Example

njk 
{% include 'smart-modal.njk' %}
html compiled 
<!-- NOTE: The modal should not be shown explicitly as below. This is for demonstration purposes only -->
<section class="smart-modal" style="display: block">
  <div class="smart-modal--content">
    <header class="smart-modal--head">
      <button class="smart-modal--close" type="button">x</button>
       <h3 class="smart-modal--title">My Modal</h3>
    </header>
    <article class="smart-modal--body">
      <p>Your content goes here!</p>
    </article>
    <footer class="smart-modal--footer">
      <button class="smart-modal--footerItem bit-btn bit-btn-clickable">Submit</button>
      <button class="smart-modal--footerItem smart-modal--close bit-btn bit-btn-clickable" type="button">Close</button>
    </footer>
  </div>
</section>

Requires

$level-100 (Number)

$theme-black (Color)

scss 
.smart-modal--content {

  max-height: 95vh;
  overflow: auto;

  @media screen and (max-width: 820px) {

  }
