Smart-X-Styles » smart-accordion

.smart-accordion

scss 
.smart-accordion {
  width: 100%;
}

Defines a responsive accordion.
*NOTE: As of now, these styles are only to support the Vue.js implementation of the Smart-Accordion. This component will not work in the example below or in the vanilla version of this framework.

Author:

  • James Stanger

Example

njk 
{% include 'smart-accordion.njk' %}
html compiled 
<section class="smart-accordion">
  <button class="smart-accordion--toggle is-expanded">Click to expand</button>
  <article class="smart-accordion--content" style="height: 58px">
     <h1>Demo!</h1>
  </article>
</section>

.smart-accordion--toggle

scss 
.smart-accordion--toggle {
  padding: 5px 20px;
  position: relative;
  z-index: 20;
  display: block;
  cursor: pointer;
  min-height: 50px;
  width: 100%;
  background: $theme-generic;
  border: none;
  text-align: left;

  &:hover {
    background: lighten($theme-generic, 5%);

    &:after {
      /* Only apply drop-arrow on hover */
      @include drop-arrow(-11px, 15px);
    }
  }

  &.is-expanded {
    /* Styles for expanded smart-accordion */
    background: saturate($theme-main, 10%);

    &:hover {
      &:after {
        /* Only apply drop-arrow on hover */
        @include drop-arrow(10px, 15px, 10px);
      }
    }
  }
}

.smart-accordion--content

scss 
.smart-accordion--content {
  overflow: hidden;
  height: 0;
  position: relative;
  z-index: 10;
  transition: height 500ms ease-in-out;
  background: lighten($theme-generic, 10%);
  box-shadow: 0 0 2px 0 inset;

  > * {
    /* Immediate children of smart-accordion--content should not have a margin so content can be resized properly. */
    margin: 0;
  }
}
